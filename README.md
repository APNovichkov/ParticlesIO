### Manual

1. Install <a href="https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html">Java SDK 8+</a> from the Oracle website
2. Download JAR executible from this github repo

```bash
# Run in Terminal :)
java -jar particlesIO.jar
```
